# Challenge: Client Patch Gate

## Category
Reverse Engineering

## Difficulty
Medium

## Story
A patched desktop client validates an unlock token before enabling debug mode.
Recover the correct token from the bytecode verifier.

## Files to give players
- `dist/ClientPatchGate.class`

## How to run
```bash
java ClientPatchGate
```

or

```bash
java ClientPatchGate "CH4KR4X2{example}"
```

## Goal format
`CH4KR4X2{...}`
