# Challenge: Ender Rev Node

## Category
Reverse Engineering

## Difficulty
Medium

## Story
A legacy service node checks an activation key using layered byte transforms.
The binary also contains an old developer compatibility parser that looks suspicious.
Recover the real flag and identify the vuln path.

## Files to give players
- `dist/EnderRevNode.class`

## How to run
```bash
java EnderRevNode
```

or

```bash
java EnderRevNode "CH4KR4X2{example}"
```

`CH4KR4X2{...}`
