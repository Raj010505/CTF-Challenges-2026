# Challenge: Stev3 Patch Lab

## Category
Reverse Engineering

## Difficulty
Medium

## Story
A legacy client has a hidden diagnostics switch.
It accepts a token and validates it through multiple transformed checks.
Reverse the class and recover the right flag.

## Files to give players
- `dist/Stev3PatchLab.class`

## How to run
```bash
java Stev3PatchLab
```

or

```bash
java Stev3PatchLab "CH4KR4X2{example}"
```

## Goal format
`CH4KR4X2{...}`
