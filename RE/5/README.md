# Challenge: POD5 Vault

## Category
Reverse Engineering

## Difficulty
Easy-Medium

## Story
A tiny Java utility protects access to a vault.
The binary accepts a single input and prints access granted/denied.
Recover the correct flag without brute force.

## Files to give players
- `dist/Pod5Vault.class`

## How to run
```bash
java Pod5Vault
```

or

```bash
java Pod5Vault "CH4KR4X2{example}"
```

## Goal format
`CH4KR4X2{...}`
